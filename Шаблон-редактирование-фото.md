# Шаблон — Редактирование фото

**Когда использовать:** изменение конкретного элемента существующего изображения при сохранении остального. Уровень 5 — самый точечный.

**Принцип Level 5:** "Edit only [target]. Keep everything else unchanged."

---

## Что описать

**Что сохранить:** структура лица, поза, пропорции тела, ключевые детали из референса

**Что изменить:** фон, освещение, одежда, стиль, цвета, среда, настроение

**Что добавить / убрать:** конкретный элемент

**Ограничения:** не менять черты лица, не перерисовывать субъект, не добавлять людей

---

## Формат промпта

**Русский:**
```
Используя референсное изображение, изменить только: [конкретный элемент].
Сохранить без изменений: [структура лица / поза / пропорции / одежда / освещение / фон / композиция — нужное].
[Описание нового элемента].
Не менять черты лица. Не перерисовывать субъект. Не добавлять лишних людей. Реалистичность.
```

**English:**
```
Using the reference image, edit only: [specific target].
Preserve unchanged: [face structure / pose / proportions / outfit / lighting / background / composition — as needed].
[Description of new element].
Do not change facial features. Do not redesign the subject. Do not add extra people. Maintain realism.
```

---

## Типовые сценарии

**Замена фона:**
```
Edit only: background environment. Keep unchanged: subject, pose, lighting on subject, proportions.
New background: [описание]. No change to face, outfit, or body. Realistic light matching.
```

**Замена одежды:**
```
Edit only: outfit. Keep unchanged: facial structure, skin tone, pose, body proportions, background.
New outfit: [описание]. Preserve identity strictly. No change to face or hair.
```

**Замена освещения:**
```
Edit only: lighting and color grading. Keep unchanged: subject, composition, pose, background.
New lighting: [описание]. Do not alter facial features or scene composition.
```

**Точечное удаление:**
```
Edit only: remove [элемент]. Keep everything else exactly as in the reference.
Do not alter surrounding areas. Realistic in-painting. No extra elements added.
```

---

## Пример (рейтинг 8/10)

**Задача:** заменить фон портрета на городской вечерний — сохранить лицо и позу

**Русский prompt:**
Используя референсное изображение: изменить только фон. Сохранить без изменений: лицо (структура, черты, тон кожи), поза, одежда, пропорции, освещение на субъекте. Новый фон: вечерняя городская улица — боке огней, мягкий синий сумерки, тёплые уличные лампы на расстоянии. Освещение на субъекте должно реалистично соответствовать вечернему городскому свету. Не менять черты лица. Не добавлять людей. Высокий реализм.

**English prompt:**
```text
Edit only: background. Preserve unchanged: face (structure, features, skin tone), pose, outfit, proportions, existing lighting on subject. New background: evening city street — bokeh lights, soft blue dusk, warm street lamps in distance. Lighting on subject must realistically match the evening urban environment. Do not change facial features. Do not add people. High realism.
```

**Ключевой принцип:** "Изменить только [X]. Сохранить без изменений: [подробный список]" — чем детальнее список сохраняемого, тем точнее редактирование.
