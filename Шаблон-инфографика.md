# Шаблон — Инфографика / Схема / Диаграмма

**Когда использовать:** инфографика, схемы процессов, диаграммы, образовательные визуалы, технические чертежи. Уровень 3.

---

## Что описать

**Тема и задача:** что объясняем, сравниваем или демонстрируем

**Структура:** количество блоков, порядок чтения, точные подписи, заголовок, легенда

**Формат:** 16:9 (презентация) / 4:5 (соцсети) / 9:16 (вертикальный) / сетка / таймлайн / флоучарт / радиальный

**Стиль:** чистый редакционный / технический / образовательный / минималистичный

---

## Формат промпта

**Русский:**
```
Инфографика на тему "[тема]": [цель — объяснить / сравнить / показать процесс].
Структура: [количество блоков/секций], последовательность [описание], заголовок "[текст]".
Подписи точно: [перечисление ключевых label'ов]. Формат [тип], стиль [описание].
Читаемая типографика, без декоративного мусора, иконки одного стиля, без лишних подписей.
```

**English:**
```
Infographic about "[topic]": goal — [explain / compare / show process].
Structure: [number of sections], reading sequence [description], title "[text]".
Exact labels: [list key labels]. Format [type], style [description].
Clean readable typography, no decorative clutter, consistent icon style, exact text only, no extra labels.
```

---

## Обязательные ограничения

```
exact text only (no additions), readable typography, no decorative clutter,
consistent icon usage, no misspellings, correct reading sequence
```

---

## Пример (рейтинг 8/10)

**Задача:** технический чертёж ювелирного украшения с видами и размерами

**Русский prompt:**
Техническая инфографика — дизайн-лист кольца: 4 вида (фронт, профиль слева, вид сверху, перспектива). Линии размеров с числами, заметки о материале (золото 18к, рубин 3мм). Белый фон, тонкие технические линии, без декора. Стиль: ювелирный конструкторский чертёж. Все подписи точно: "Front view", "Side view", "Top view", "Isometric". Без случайных символов, без декоративных элементов, читаемый технический шрифт.

**English prompt:**
```text
Technical infographic — jewelry design sheet for a ring: 4 views (front, left profile, top, isometric perspective). Dimension lines with numbers, material notes (18k gold, 3mm ruby). White background, fine technical lines, zero decoration. Style: jewelry engineering blueprint. Exact labels: "Front view", "Side view", "Top view", "Isometric". Manufacturable construction shown. No random symbols, no decorative elements, clean technical typography.
```

**Ключевой принцип:** Вместо "красивые украшения" — запрос формата "дизайн-лист с видами + размерными линиями + нотами о материале" даёт точный технический результат.
