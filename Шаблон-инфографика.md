# Шаблон: инфографика / диаграмма / схема

Используй для инфографик, схем, диаграмм, образовательных карточек, process visuals.

## Что нужно описать

Тема:
- о чем инфографика
- цель: объяснить, сравнить, показать процесс

Структура:
- количество блоков
- порядок чтения
- exact labels
- заголовок и подписи

Layout:
- 16:9 presentation
- 4:5 social post
- vertical 9:16
- grid
- timeline
- flowchart
- radial diagram

Стиль:
- clean editorial
- technical
- educational
- minimal
- brand-like

Ограничения:
- exact text only
- no extra labels
- no misspelled words
- clean readable typography
- no decorative clutter
- consistent icon style

## Рабочий формат

Русский prompt:
Инфографика на тему [тема], структура [количество блоков/шагов], точные подписи [список], layout [тип], стиль [какой], [ограничения].

English prompt:
Create a clean infographic about [topic]. Structure: [sections/steps]. Exact labels: [labels]. Layout: [layout]. Style: [style]. Constraints: exact text only, readable typography, no extra labels, no clutter.
