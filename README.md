# PromptHub — система создания промптов для генерации изображений

**PromptHub** — это профессиональная система prompt-инжиниринга для нейросети **Nano Banana 2**, работающая через интерфейс **web-Gemini**. Система позволяет превращать идеи на естественном языке в готовые production-промпты на русском и английском.

---

## Что умеет система

- Принимает запрос на любом языке в свободной форме — без специальных команд
- Задаёт 1–3 уточняющих вопроса только при необходимости
- Выдаёт готовый промпт сразу на **русском и английском**
- Поддерживает итерации: v2, v3 на основе обратной связи
- Ведёт журнал экспериментов и успешных промптов
- Работает с референсами, сериями, брендовой последовательностью

## Как работает пайплайн

```
Запрос пользователя
        ↓
1. Маршрутизация намерения (генерация / редактирование / итерация / критика)
2. Проверка памяти (прошлые эксперименты, успешные формулы)
3. Выбор уровня промпта (1–5)
4. Оценка полноты данных
5. Уточняющие вопросы (1–3, только если нужно)
6. Приоритет референсов
7. Сборка промпта
8. Проверка анти-паттернов
9. Финальный контроль качества
        ↓
Промпт на русском + английском
        ↓
Генерация в web-Gemini → Обратная связь → v2/v3
```

## Уровни сложности промпта

| Уровень | Когда использовать |
|---------|-------------------|
| **Level 1** — Quick | Простая идея, низкий риск |
| **Level 2** — Production | Коммерческий, fashion, beauty, product, poster |
| **Level 3** — Structured | Сложные сцены, несколько объектов, текст в кадре |
| **Level 4** — High-Risk Lock | Конкретные люди, продукт, серийные персонажи |
| **Level 5** — Local Edit | Точечная правка одного элемента, всё остальное без изменений |

## Формат вывода

Каждый промпт выдаётся в стандартном формате:

```
**Русский prompt:**
[детальный промпт на русском]

**English prompt:**
[готовый к копированию промпт для Nano Banana 2]

**Следующий шаг:** [конкретный совет]
```

## Быстрый старт

1. Откройте `AGENTS.md` — главные инструкции для агента
2. Используйте `INDEX.md` как карту навигации
3. Опишите задачу свободным языком — система определит тип и подберёт шаблон
4. Получите промпт и вставьте в web-Gemini (Nano Banana 2)
5. Дайте обратную связь — система запишет результат и предложит v2

## API-интеграция (опционально)

Локальный запуск через Nano Banana API доступен только при явном запросе. Настройка через `.env` файл (ключи никогда не хранятся в коде).
