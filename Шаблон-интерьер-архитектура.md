# Шаблон — Интерьер / Архитектура

**Когда использовать:** визуализация пространств — квартиры, рестораны, отели, офисы, архитектурные объекты. Уровень 2.

---

## Что описать

**Пространство:** тип (жилое / коммерческое / общественное), стиль, материалы, планировка, главный визуальный акцент

**Материалы:** дерево, камень, металл, стекло, ткань, бетон, мрамор

**Свет:** естественный оконный / тёплые лампы / архитектурная подсветка / вечер / сбалансированная экспозиция

**Композиция:** широкий интерьерный план / угловой вид / симметрия / уровень глаз / чистые перспективные линии

---

## Формат промпта

**Русский:**
```
[Тип пространства] в стиле [стиль], материалы [перечисление].
Главный акцент — [ключевой элемент]. [Освещение].
[Ракурс]: [тип кадра], чёткие перспективные линии, вертикали строго прямые.
[Атмосфера]. Без деформированной архитектуры, без невозможной геометрии, без захламлённого декора.
```

**English:**
```
[Space type] in [style], materials: [list].
Main visual focus — [key element]. [Lighting].
[Camera angle]: [frame type], clean perspective lines, strictly vertical walls.
[Atmosphere]. No warped architecture, no impossible geometry, no cluttered decor.
```

---

## Обязательные ограничения

```
no warped architecture, no impossible geometry, no cluttered decor,
straight vertical lines, realistic scale, no floating objects
```

---

## Пример (рейтинг 8/10)

**Задача:** современная гостиная в скандинавском стиле, дневной свет

**Русский prompt:**
Современная гостиная в скандинавском стиле: светлое дерево, белые стены, серый лён. Главный акцент — большое панорамное окно с видом на зелёный двор. Натуральный дневной свет — мягкий и рассеянный, без резких теней. Угловой вид из левого переднего угла, кадр в ширину. Диван, кофейный столик, напольное растение — минимализм без лишних предметов. Чёрные акценты в светильниках и рамах. Вертикали строго прямые, перспектива корректная, без деформаций. Тёплая-нейтральная цветовая палитра.

**English prompt:**
```text
Modern living room in Scandinavian style: light wood, white walls, grey linen. Main focal point — large panoramic window with view of green courtyard. Natural daylight — soft and diffused, no harsh shadows. Corner view from front-left, wide framing. Sofa, coffee table, floor plant — minimalism with no clutter. Black accents on light fixtures and frames. Strictly straight verticals, correct perspective, no distortion. Warm-neutral color palette.
```

**Ключевой принцип:** Угловой вид + явное указание "вертикали строго прямые" предотвращает типичную деформацию архитектурной геометрии.
