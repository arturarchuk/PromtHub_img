# Протокол улучшения v2 / v3

Используй этот файл, когда пользователь говорит, что результат плохой, странный, "почти ок, но..." или просит v2/v3.

## Быстрый workflow

1. Отдели ошибку генерации от намеренного пожелания пользователя.
2. Выбери 2-5 тегов по `Memory-Tags.md`.
3. Запиши короткий урок в `Журнал-экспериментов.md` по `Правила-обратной-связи.md`.
4. Сохрани рабочие элементы: лицо, композицию, свет, стиль, товар или текст, если пользователь просит.
5. Добавь точный fix-фрагмент из таблицы ниже.
6. Выдай новый prompt в стандартном формате RU + EN.

## Таблица fixes

| Проблема | Вероятная причина | Готовый fix-фрагмент для prompt |
|----------|-------------------|----------------------------------|
| Лицо исказилось | identity не зафиксирована, слишком много style pressure | `Preserve the same facial structure, eye shape, nose, lips, jawline, cheekbones, face width, skin tone, and head proportions. Do not distort or beautify the identity.` |
| Лицо пластиковое | over-retouching, beauty filter, CGI-лексика | `natural realistic skin micro-texture, visible pores, subtle imperfections, no plastic retouching, no airbrushed smoothing, no doll-like skin` |
| Глаза неживые | нет catchlights, слишком гладкая обработка | `natural catchlights in the eyes, relaxed eyelids, focused gaze, no glassy eyes, no dead stare` |
| Руки странные | руки крупно/сложно поставлены без контроля | `natural hand anatomy, relaxed fingers, anatomically correct wrists, no extra fingers, no fused fingers, no distorted hands` |
| Руки не важны | модель пытается показать их крупно | `hands relaxed and partially outside the frame / hands naturally resting under the table, not emphasized` |
| Слишком темно | underexposed shadows, low-key без fill | `balanced exposure, soft visible key light, gentle fill light, important details clearly visible, no crushed black shadows` |
| Слишком ярко | overexposure, белый клиппинг | `controlled highlights, natural contrast, no blown-out whites, no overexposed skin, preserve texture in bright areas` |
| Дешёвый вид | переизбыток глянца, случайный декор | `restrained luxury styling, premium materials, fewer decorative props, controlled contrast, no oversaturated colors, no cheap glossy look` |
| Фон перегружен | слишком много props и случайных деталей | `clean uncluttered background, fewer props, clear subject separation, softly blurred secondary details, no random objects` |
| Товар исказился | product reference слабее style reference | `preserve exact product shape, proportions, logo placement, packaging geometry, material finish; no warped product, no redesign, no duplicated product` |
| Логотип поплыл | текст/логотип не закреплён | `logo placement unchanged, exact readable label, no fake letters, no stretched typography, no random extra markings` |
| Текст неправильный | не указан exact text и запреты | `Render the exact text "[TEXT]" only. Clean readable typography, no extra letters, no misspelled words, no random text, no watermark.` |
| Лишний текст | модель добавляет псевдонадписи | `no text, no letters, no numbers, no watermark, no signage, no random typography` |
| Стиль не тот | prompt смешал несколько эстетик | `Use only [target style]. Remove conflicting aesthetics. Keep lighting, palette, materials, and composition aligned with [target style].` |
| Слишком cinematic | cinematic-лексика не подходит задаче | `clean commercial photography, neutral realistic lighting, no dramatic movie lighting, no excessive color grading` |
| Еда выглядит пластиковой | нет натуральных texture cues | `realistic food texture, natural moisture, appetizing surface detail, no plastic shine, no artificial colors` |
| Интерьер кривой | перспектива не зафиксирована | `straight vertical lines, realistic architectural perspective, natural room scale, no warped walls, no bent furniture` |
| Референсы смешались | нет role/priority order | `Use Image A as [role]. Use Image B only as [role]. Image B is not an identity/product reference. Ignore [irrelevant elements] from Image B.` |
| Серия потеряла консистентность | нет anchors | `Keep the same character/product/style anchors across all frames: [anchors]. Change only [per-frame variable].` |
| Накопились артефакты | слишком длинная цепочка edits | `Using the best current image as the new reference, recreate the scene cleanly. Preserve [working elements]. Improve [issues]. Avoid accumulated artifacts and conflicting old instructions.` |

## Как писать v2

| Ситуация | Стратегия |
|----------|-----------|
| Ошибка локальная | `local edit`: менять только проблемную область, всё остальное сохранить |
| Ошибка системная | пересобрать полный prompt с ясной композицией, светом, стилем и constraints |
| Drift лица или товара | подключить `Reference-Consistency-Protocol.md` и role/priority lock |
| 3-5 итераций ухудшили результат | consolidated restart от лучшей версии |

## Conversational refinement

Исправляй маленькими точными изменениями.

Формула:

`Keep [what worked] unchanged. Improve only [problem area] by [specific fix].`

Если пользователь говорит "фон слишком темный", не меняй лицо, одежду, позу и стиль без причины.

## Запись в журнал

Формат короткой записи:

```text
### YYYY-MM-DD — [краткое название]
Сцена:
Теги:
Проблема:
Вероятная причина:
Фикс для v2:
Правило:
```

Не вставляй полный prompt в журнал, если пользователь не попросил.
