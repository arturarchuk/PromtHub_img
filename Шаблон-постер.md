# Шаблон — Постер / Баннер / Thumbnail

**Когда использовать:** рекламные постеры, обложки, thumbnail'ы, product cards, баннеры. Уровень 2–3.

---

## Что описать

**Главная идея:** что рекламируется, главный объект, желаемый эмоциональный эффект

**Композиция:** расположение объектов, зона для текста, масштаб кадра, визуальная иерархия

**Дизайн:** стиль, цветовая палитра, контраст, типографика, чистота компоновки

**Текст:** точные фразы в кавычках — или явный запрет текста если не нужен

---

## Формат промпта

**Русский:**
```
Постер/thumbnail для [тема], главный объект [субъект], композиция [расположение].
Стиль [описание], цветовая палитра [цвета], контраст [высокий/мягкий].
[Текст: "точная фраза" / без текста]. Чистая иерархия, без лишних элементов,
без водяных знаков, без случайного текста.
```

**English:**
```
[Poster/thumbnail] for [theme], main subject [subject], composition [layout].
Style [description], color palette [colors], [high/soft] contrast.
[Text: "exact phrase" / no text]. Clean visual hierarchy, no clutter,
no watermarks, no random text.
```

---

## Обязательные ограничения

```
exact text only (no additions, no spelling errors), no watermarks,
no cluttered layout, clean information hierarchy, no random background text
```

---

## Пример (рейтинг 8/10)

**Задача:** постер для фешн-коллекции — минимализм, акцент на силуэте

**Русский prompt:**
Минималистичный фешн-постер: женский силуэт в длинном пальто-оверсайз телесного цвета, стоит боком, взгляд в сторону. Чистый молочно-белый фон, естественный рассеянный свет без теней. Вертикальный формат 4:5, субъект занимает 70% кадра, нижняя треть свободна для текста. Стиль: современный редакционный минимализм. Без текста, без пропсов, без лишних элементов. Лицо без дисторсии.

**English prompt:**
```text
Minimalist fashion poster: female silhouette in long oversized coat in nude tone, standing sideways, gaze directed away. Clean milky-white background, natural diffuse light with no harsh shadows. Vertical format 4:5, subject fills 70% of frame, lower third clear for text placement. Style: contemporary editorial minimalism. No text, no props, no extra elements. No facial distortion.
```

**Ключевой принцип:** Явная резервная зона для текста в промпте ("нижняя треть свободна") предотвращает конфликт типографики с субъектом.
