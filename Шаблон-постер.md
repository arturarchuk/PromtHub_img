# Шаблон: постер / thumbnail / баннер

Используй этот шаблон для обложек, постеров, афиш, YouTube thumbnail, рекламных баннеров, карточек товара.

## Что нужно описать

Главная идея:
- что рекламируем или показываем
- главный объект
- эмоция или эффект

Композиция:
- где главный объект
- где место для текста
- крупный или общий план
- визуальная иерархия

Дизайн:
- стиль
- цвета
- контраст
- типографика, если нужен текст
- чистота композиции

Текст:
- если нужен текст, укажи точную фразу
- если текст не нужен, явно запрети текст

Ограничения:
- text must be spelled exactly as provided
- no extra random letters
- no watermark
- no cluttered layout
- clean information hierarchy

## Короткий рабочий формат

Русский prompt:
Постер/thumbnail для [тема], главный объект [что], композиция [как расположено], стиль [какой], цвета [какие], текст "[точная фраза]" если нужен, [ограничения].

[same idea in English, optimized for a clean graphic result]
