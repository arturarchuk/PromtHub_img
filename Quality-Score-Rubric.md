# Quality Score Rubric

Этот файл помогает оценивать результат генерации не общими словами, а по критериям.

## Общая оценка

Оценка 1-10:

- 1-3: результат не попал в задачу, нужен clean restart
- 4-5: есть отдельные рабочие элементы, но prompt нужно пересобрать
- 6-7: основа работает, нужна v2
- 8: хороший результат, нужны точечные правки
- 9: почти финал, сохранить формулу
- 10: сохранить как удачный prompt

## Критерии

Оцени только релевантные критерии. Для инфографики не оценивай лицо, для портрета без текста не оценивай типографику.

| Критерий | 0 | 1 | 2 | Заметки |
|----------|---|---|---|---------|
| Идея | не попала | частично | точно | |
| Главный объект | не читается | читается с трудом | читается сразу | |
| Композиция | хаос | рабочая, но слабая | чистая иерархия | |
| Свет | ломает результат | приемлемый | усиливает задачу | |
| Реализм / стиль | не тот стиль | близко | точно по задаче | |
| Лицо / identity | не похоже / искажено | частично | стабильно и живо | |
| Руки / анатомия | ошибка | терпимо | естественно | |
| Товар | форма/логотип сломаны | частично | точное сохранение | |
| Текст | неверный | частично читаем | точный и читаемый | |
| Референсы | роли смешались | частично | роли соблюдены | |
| Overall polish | дешево/грязно | средне | production-ready | |

**Итоговая логика:**

| Сумма / впечатление | Следующий шаг |
|---------------------|---------------|
| 0-8 или результат не в задачу | `clean restart` |
| 9-14, есть рабочие элементы | пересобрать prompt от лучших элементов |
| 15-19 | `v2` с 1-3 точными fixes |
| 20-21 или почти финал | `local edit` / финальная полировка |
| 22 и выше / 9-10 из 10 | сохранить в `Gallery.md` и, если prompt доступен, в `Мои-удачные-промты.md` |

### Идея

- Понятно ли, что изображение показывает?
- Совпадает ли с намерением пользователя?
- Не ушла ли модель в другую тему?

### Главный объект

- Главный объект читается сразу?
- Не конкурирует ли фон?
- Не слишком ли мелкий объект?

### Композиция

- Правильный план и ракурс?
- Есть ли чистая визуальная иерархия?
- Не перегружен ли кадр?

### Свет

- Свет подходит задаче?
- Видны ли важные детали?
- Нет ли плоского, дешевого или слишком драматичного света?

### Реализм / стиль

- Стиль соответствует задаче?
- Нет ли пластиковости, рендера там, где нужно фото?
- Нет ли случайной cinematic-стилизации?

### Лицо / identity

Используй для портретов и референсов:

- лицо похоже на референс?
- не изменилась ли геометрия?
- живые ли глаза?
- натуральная ли кожа?
- не изменился ли ракурс головы?

### Руки / анатомия

- естественные ли руки?
- нет ли лишних пальцев?
- нет ли дублированных рук?
- можно ли спрятать руки, если они не важны?

### Товар

- форма не искажена?
- логотип/упаковка сохранены?
- материал выглядит правдоподобно?
- товар не продублирован?

### Текст

- текст написан точно?
- нет ли лишних букв?
- читаема ли типографика?
- не конкурирует ли текст с объектом?

### Референсы

- каждая роль референса соблюдена?
- не смешалась ли identity?
- стиль не изменил товар или лицо?
