# Уровни промпта

Этот файл помогает выбрать степень детализации. Не каждый запрос требует огромного production prompt.

## Level 1: Quick prompt

Используй, когда идея простая, риск низкий, референсов нет.

Примеры:
- девушка в дорогом ресторане ночью
- черные часы на мраморе
- город будущего под дождем

Что включить:
- главный объект
- место или фон
- композиция
- стиль
- свет
- aspect ratio
- 2-4 релевантных constraints

Не перегружай prompt техническими деталями.

## Level 2: Production prompt

Используй для коммерческих, красивых, премиальных, fashion, beauty, product, cinematic и poster задач.

Что включить:
- четкий главный объект
- визуальная иерархия
- свет и цвет
- материалы и фактуры
- композиция и кадрирование
- aspect ratio
- точные constraints против типичных ошибок

Это основной рабочий уровень PromptHub.

## Level 3: Structured prompt

Используй, когда есть много требований:
- несколько объектов
- точный текст
- сложная сцена
- инфографика
- серия изображений
- несколько референсов
- важная consistency

Структурируй prompt блоками:
- Subject
- Composition
- Lighting
- Style
- Preserve
- Change
- Constraints

## Level 4: High-risk lock prompt

Используй, когда нельзя допустить drift:
- конкретное лицо
- товар с формой, упаковкой или логотипом
- исторический костюм
- серия с одним персонажем
- несколько конфликтующих референсов

Обязательные блоки:
- role of each reference
- priority order
- identity/product locks
- what must be ignored from lower-priority references
- negative constraints against drift

## Level 5: Local edit prompt

Используй для точечной правки изображения.

Главная формула:
Edit only [target area]. Keep everything else unchanged.

Обязательно указать:
- область редактирования
- что изменить
- что сохранить
- масштаб и материал новых деталей
- запрет global redraw

## Быстрое правило выбора

- Просто и понятно -> Level 1.
- Коммерческое или красивое -> Level 2.
- Много требований -> Level 3.
- Лицо, товар, референсы, серия -> Level 4.
- Исправить часть изображения -> Level 5.

Если сомневаешься, выбирай меньший уровень и добавляй только те детали, которые реально снижают риск.
