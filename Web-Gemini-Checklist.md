# Web Gemini Checklist

Этот файл помогает пользователю правильно запускать prompt в веб-Gemini / Nano Banana 2.

## Перед генерацией

Проверь:

- открыт веб-Gemini
- выбран инструмент Create images
- выбран нужный режим: Fast / Thinking / Pro, если доступно
- English prompt скопирован полностью
- aspect ratio указан в prompt, если важен
- если нужен текст на изображении, точная фраза стоит в кавычках
- если текст не нужен, есть no text / no letters / no watermark

## Если есть референсы

Проверь:

- все нужные изображения загружены
- референсы загружены до отправки prompt
- понятно, какой референс за что отвечает
- identity reference содержит хорошо видимое лицо
- product reference показывает форму, логотип и упаковку
- outfit/style reference не должен влиять на лицо, если там другой человек

## Для портрета

Проверь:

- лицо достаточно крупно
- свет не закрывает глаза
- руки не выведены крупно без необходимости
- указан natural skin texture, если нужен реализм
- если есть identity, загружен reference image

## Для товара

Проверь:

- товар главный объект
- указаны форма, материал, поверхность
- если есть логотип, загружен референс
- есть constraints: no warped product shape, no redesign, no duplicated objects

## Для постера / текста

Проверь:

- точный текст в кавычках
- задана иерархия текста
- есть место для текста
- есть constraints: no extra letters, no misspelled words, clean readable typography

## После генерации

Пришли в PromptHub:

1. результат изображения
2. что нравится
3. что не нравится
4. что сохранить
5. что изменить
6. оценку близости 1-10

## Если результат почти хороший

Пиши feedback точечно:

- сохранить композицию
- сохранить лицо
- сохранить свет
- исправить только руки
- сделать фон дороже
- убрать лишний текст

## Если результат совсем не тот

Пиши:

- что именно неверно
- какая цель была на самом деле
- делать v2 или clean restart
