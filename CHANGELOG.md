# CHANGELOG

## [2026-05-14] Глобальная реструктуризация системы

**Изменено:**
- `AGENTS.md` — переписан заново: объединяет содержимое бывших `Главное-правило.md` и `README-PromptHub.md`. Теперь единственный источник правил для агента.
- `INDEX.md` — переделан из списка файлов в дерево решений с маршрутизацией по типу задачи.
- Все шаблоны (`Шаблон-*.md`) — добавлены разделы `## Пример` с реальными кейсами.

**Добавлено:**
- `CONTRIBUTING.md` — правила командной работы: как добавлять записи, обновлять шаблоны, именовать файлы.
- `CHANGELOG.md` — этот файл, история изменений.
- `Gallery.md` — таблица результатов с оценками для отслеживания лучших промптов.
- `API-Guide.md` — объединённый гайд по API (заменяет три отдельных файла).

**Удалено:**
- `Главное-правило.md` — содержимое перенесено в `AGENTS.md`.
- `README-PromptHub.md` — содержимое перенесено в `AGENTS.md`.
- `Nano-Banana-API-Workflow.md`, `Google-Cloud-Nano-Banana-Guide.md`, `API-Test-Quickstart.md`, `Web-Gemini-Checklist.md` — объединены в `API-Guide.md`.

**Почему:** репозиторий рос органически и накопил дублирование. 4 файла описывали одни и те же правила агента, INDEX не помогал навигации, шаблоны не содержали примеров. Реструктуризация устраняет эти проблемы.
