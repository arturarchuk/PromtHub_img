# Форматы и соотношения сторон

Используй этот файл, когда пользователь просит изображение для конкретной платформы или назначения.

## Универсальные форматы

1:1 - квадрат, удобно для аватаров, товаров, Instagram grid, карточек.

4:5 - вертикальный пост, портрет, fashion, Instagram feed.

9:16 - stories, Reels, TikTok, вертикальные обложки, phone wallpaper.

16:9 - YouTube thumbnail, баннер, широкий cinematic frame, презентации.

3:2 - фотографический кадр, lifestyle, travel, editorial.

2:3 - постер, афиша, книжная обложка.

## Рекомендации по типам задач

Портрет:
- 4:5 для портрета
- 9:16 для сторис или full body
- 1:1 для аватара

Товар:
- 1:1 для маркетплейса
- 4:5 для рекламы и соцсетей
- 16:9 для баннера

Постер:
- 2:3 для афиши
- 4:5 для соцсетей
- 16:9 для thumbnail

Кинематографичная сцена:
- 16:9 для кадра из фильма
- 9:16 для вертикальной мобильной сцены
- 3:2 для editorial/photo look

Инфографика:
- 16:9 для презентации
- 4:5 для соцсетей
- 1:1 для компактной схемы

## Как писать в prompt

Добавляй в конец:

aspect ratio 4:5
vertical 9:16 composition
wide 16:9 cinematic frame
square 1:1 product composition

## Если пользователь не указал формат

Выбери сам:

- портрет человека: 4:5
- товар: 1:1 или 4:5
- постер: 2:3 или 4:5
- кино-сцена: 16:9
- сторис/мобильный контент: 9:16
